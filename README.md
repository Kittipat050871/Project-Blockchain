🎬 บทนำ: เตรียมกระเป๋าตังค์ (MetaMask)
ก่อนเริ่ม ให้คุณเปิด MetaMask แล้วตั้งชื่อกระเป๋าไว้ 2 ใบให้จำง่ายๆ ครับ:

Account 1 (Landlord): เอาไว้เป็นเจ้าของหอ

Account 2 (Tenant): เอาไว้เป็นผู้เช่า

🏗️ ฉากที่ 1: สร้างสัญญา (Remix)
ขั้นตอนนี้คือการสร้าง "กฎหมาย" ลงบนบล็อกเชน (ทำครั้งเดียวตอนผู้เช่าเพิ่งย้ายเข้า)

เปิดเว็บ Remix และเปิดไฟล์ DormBillV2.sol

สลับ MetaMask ไปใช้กระเป๋า Landlord

กด Compile ให้ผ่าน (ติ๊กถูกสีเขียว)

ไปที่เมนู Deploy (แถบที่ 4 ซ้ายมือ) ทำการตั้งค่าก่อนกด Deploy ดังนี้:

VALUE: ใส่ 5000 และเปลี่ยนหน่วยข้างหลังเป็น wei (นี่คือการจำลองการล็อกเงินประกัน)

_tenant: ก๊อปปี้ Address ของกระเป๋าผู้เช่ามาวาง

_monthlyRent: ใส่ยอดค่าเช่าเหมาจ่าย เช่น 4850

_waterRate: 15

_electricRate: 8

_contractHash: ใส่ 0x123 (หรืออะไรก็ได้เพื่อเป็น Mock data)

กด Deploy และกดยืนยันใน MetaMask

สำคัญ: ก๊อปปี้ Contract Address อันใหม่ล่าสุด ที่ได้มา เก็บไว้!

💻 ฉากที่ 2: อัปเดตโค้ด (React)
กลับมาที่ VS Code เปิดไฟล์ App.js

เอา Contract Address ที่เพิ่งก๊อปปี้มา วางทับในบรรทัดที่ 6:
const contractAddress = "0xเลขใหม่ของคุณ";

กด Save (Ctrl+S) แล้วเปิดหน้าเว็บ localhost:3000 (กด F5 รีเฟรช 1 รอบ)

🏢 ฉากที่ 3: สวมบท "เจ้าของหอ" (จดมิเตอร์)
ที่หน้าเว็บ ให้เช็คว่าเชื่อมต่อ MetaMask ด้วย กระเป๋า Landlord (มุมขวาบนจะขึ้นว่า "ผู้ดูแลหอพัก")

ในแผง "ตั้งค่าข้อมูลห้อง" ลองพิมพ์เลขห้อง (เช่น 101), เลือกตึก, เลือกประเภทห้อง

ในแผง "บันทึกมิเตอร์" กรอกตัวเลขสมมติ:

ค่าน้ำ: ก่อน 100 / หลัง 105 (ระบบจะคำนวณว่าใช้ไป 5 หน่วย)

ค่าไฟ: ก่อน 2000 / หลัง 2150 (ระบบจะคำนวณว่าใช้ไป 150 หน่วย)

เล็งไปที่กล่องขวาล่างสุด กดปุ่ม "สร้างบิล (Generate Bill)"

กดยืนยันใน MetaMask แล้วรอจนมี Alert เด้งว่า "สร้างบิลสำเร็จ"

🧑‍💻 ฉากที่ 4: สวมบท "ผู้เช่า" (จ่ายเงิน)
กดเปิด MetaMask แล้วสลับไปใช้กระเป๋า Tenant

หน้าเว็บจะเปลี่ยนบทบาทให้อัตโนมัติ (มุมขวาบนจะขึ้นว่า "ผู้เช่า ห้อง 101")

คุณจะเห็นบิลเรียกเก็บเงินโชว์ขึ้นมา! มีแจกแจงค่าน้ำ ค่าไฟ และยอดรวมที่ต้องจ่าย (ตรงกับที่ทดสอบใน Remix)

มีกล่อง Escrow โชว์ว่าตอนนี้คุณมีเงินประกันฝากไว้ 5000 wei

กดปุ่มสีเขียว "💸 ชำระเงินผ่าน Smart Contract"

กดยืนยันการจ่ายใน MetaMask แล้วรอจน Alert เด้งว่า "ชำระเงินสำเร็จ"

บิลจะหายไป และเปลี่ยนเป็นข้อความ "🎉 ไม่มียอดค้างชำระ"

🛑 ฉากที่ 5: ปิดจบสัญญา (คืนเงินประกัน)
จำลองสถานการณ์ว่าผู้เช่าเรียนจบ จะย้ายออกแล้ว

สลับ MetaMask กลับไปเป็น Landlord

เล็งไปที่กล่องสีแดงๆ ทางขวาบนที่เขียนว่า "🔒 สิ้นสุดสัญญาเช่า"

สมมติว่าผู้เช่าทำประตูพังนิดหน่อย ให้กรอกค่าเสียหายไป 1000 (หน่วย wei)

กดปุ่ม "ปิดสัญญา" แล้วกดยืนยันใน MetaMask

ระบบจะหักเงิน 1,000 เข้ากระเป๋า Landlord และโอนเงินประกันที่เหลืออีก 4,000 คืนเข้ากระเป๋า Tenant ให้อัตโนมัติ!

ระบบจะล็อก ไม่ให้ออกบิลหรือทำอะไรกับสัญญาฉบับนี้ได้อีก เป็นอันจบกระบวนการ 100% ครับ!

💡 ทริคสำหรับการพรีเซนต์: ถ้าอยากโชว์ "ป้ายเตือนค่าปรับล่าช้าสีแดงๆ" ให้ดู อย่าลืมเข้าไปแก้โค้ดใน Remix ตรง generateBill ให้เป็น dueDate = block.timestamp - 10 days; ก่อน Deploy สัญญาใหม่นะครับ ระบบจะได้โดนหลอกว่าเกินกำหนดแล้วตั้งแต่วินาทีแรกที่ออกบิลครับ

🚀 วิธีรัน Local Server (localhost:3000)
ขั้นตอนที่ 1: เปิด Terminal ใน VS Code

เปิดโปรเจคหอพักของคุณในโปรแกรม VS Code

ไปที่เมนูด้านบนสุด เลือก Terminal -> New Terminal (หรือใช้คีย์ลัด: กดปุ่ม Ctrl + ` (ปุ่มเปลี่ยนภาษา) บนคีย์บอร์ด)

ขั้นตอนที่ 2: พิมพ์คำสั่งเริ่มระบบ

ในหน้าต่าง Terminal ด้านล่าง ให้เช็คก่อนว่าบรรทัดคำสั่งอยู่ในโฟลเดอร์โปรเจคของคุณแล้ว

พิมพ์คำสั่งนี้ลงไป แล้วกด Enter:

Bash
npm start
ขั้นตอนที่ 3: รอระบบทำงาน

ระบบจะทำการประมวลผล (Compiling) สักครู่

เมื่อเสร็จแล้ว มันจะเด้งเปิดเบราว์เซอร์ (Chrome, Edge ฯลฯ) หน้าเว็บ http://localhost:3000 ขึ้นมาให้อัตโนมัติครับ

🛑 วิธียกเลิก / ปิด Server
ถ้าต้องการหยุดการทำงานของ Server (เช่น จะลง Library เพิ่ม หรือคอมค้าง)

คลิกที่หน้าต่าง Terminal ใน VS Code

กดปุ่ม Ctrl + C (กดพร้อมกัน)

ระบบจะถามว่า Terminate batch job (Y/N)? ให้พิมพ์ Y แล้วกด Enter ครับ
